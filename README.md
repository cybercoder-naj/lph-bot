# Los Patos Hermanos

#LPH4Life

## Features

This is a discord that does:
  - [x] Looks for Endurance Races on Simgrid and create channels for them. Also deletes the channels when the race is over.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/cybercoder-naj/lph-bot.git
   cd lph-bot
   ```
2. Install the required dependencies:
   ```bash
    pnpm install
   ```
3. Create a `.env` file in the root directory and add your Discord bot token:
   ```env
   cp .env.template .env
   ```
4. Start the bot:
   ```bash
   pnpm start
   ```
